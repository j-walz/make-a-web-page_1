Notes From Stage0

How the Internet Works

The internet is the worlds largest network of computers. Basically it is just a web for computers
to talk to other computers, but on a giant scale. The user will get on a browser and send a request
over the internet to a web server that is hosting what ever information the user wants. The web server
then repsonds to the user with the information request and sends it back across the internet to appear on the user's browser.

HTML

HTML is the basic language of the internet. Almost all of the web pages on the internet are written using HTML.
HTML documents can contain links, images, text, video, and much more. The way that these different elements
of an HTML document are written into the document are with tags. Tags are markers that notify the HTML document
that a specified set of text is not just text but an insrtuction of how to treat other sets of text. Tags are surrounded by < and >
with the closing tag having a / infront of it. This is so the HTML document knows how to treat that text, and then the tags
themselves surround the bit of text that they are affecting. An example of a tag is the b tag, which tells the HTML document
that all of the text inbetween the opening and closing b tags should be bold.

Together a tag and the text that thattag is affecting are called and element. There are two types of elements, inline
and block. Inline elements keep the element within the line of text and do not add anything extra to the text.
Block elements put an invisible box around the text that the writer can use to set that bit of text apart from others.

Thinking Like a Computer

Computers do exactly what they are told, but what they are told is not always what is meant. To have think like
a computer is to be able to think in systematic steps. One must take everything one step at a time and without
assuming any intetnion that may seemt to be built into the intrusction. This can be fairly difficult and requires much practice.

The Document Object Model

The Document Object Model, or DOM, is how web browsers interperit html files like this one. An html page is not written in some bizaar language, rather it is just written using a particular structure that allows browsers to understand what the author intends. The DOM is a branched structure, having elements branch to other elements which may in turn branch off further and further. Not only does this allow for browers to read html documents it is also fairly easliy understood by the human eye.

Developer Tools

The broswer Google Chrome has a usefull feature in the Developer Tools. They allow the user to inspect the elements of what ever web page he or she is viewing. If a person who is starting out in fron end web development is stuck or sees an interesting feature on a web page, than that person can utilize the Developer Tools to see what kind of tag, css, and or java script was used to produce the desired effect.

Text Editors

Text Editors are a great tool for programmers. They help the programmer by keeping him or her in line with the DOM and grammatical structure of what ever language is being used

HTML and Boxes

When one is looking at a web page, especially a good one, many shapes and design features will become apparent. However, looks may be deceiving. All of the elements display on an HTML page are actually boxes. Though one may see a circle or a triangle, those shapes are contained in boxes. This makes it much easier for prgrammers to plan and lay out a web page as well as rearrange it if neccesary.

Avoid Repition

Writing a program or creating a web page can be very time consuming and tedious. Becuase of that fact it is important for programmers to avoid repition as much as possible so as not to add to their already burdensome task. In avoiding rewriting the same code over and over consistency and the ability to change large portions of code are gained. By reusing a trusted piece of code programmers can be assuered that what they are writing will be good.

CSS

CSS, or Cascading Style Sheets, is the language used to add style to HTML web pages. With CSS a programmer can take a bland and uniteresting web page and turn it into a colorful and attention grabbing web page.
